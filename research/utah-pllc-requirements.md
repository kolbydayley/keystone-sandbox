# Utah PLLC Requirements for Nurse Practitioner

**Researched:** 2026-01-18
**Status:** Complete

## Summary

Utah requires a **PLLC** (Professional Limited Liability Company) for licensed professionals like nurse practitioners. This is essentially an LLC with additional requirements for licensed professionals.

## Formation

| Item | Details |
|------|---------|
| Filing Fee | **$59** |
| Where to File | [corporations.utah.gov](https://corporations.utah.gov) |
| Name Requirement | Must include "Professional Limited Liability Company" or "PLLC" or "P.L.L.C." |
| Registered Agent | Required (must have Utah address) |
| Operating Agreement | Recommended but not required by state |

## Steps to Form

1. **Choose name** — Must include PLLC designation and comply with nursing board naming rules
2. **Appoint registered agent** — Must have Utah physical address
3. **File Certificate of Organization** — Online at businessregistration.utah.gov ($59)
4. **Create operating agreement** — Not filed with state but recommended
5. **Get EIN** — From IRS (free) for tax purposes

## Ongoing Requirements

| Item | Details |
|------|---------|
| Annual Report | Due on anniversary of formation |
| Annual Fee | $18 (mail) or $23 (online) |
| Singed Agent | Must maintain continuously |

## Utah NP Practice Notes

- Utah grants NPs **independent practice authority**
- Some restrictions on prescribing certain medications
- No collaborative agreement required for general practice

- **Total time to operational:** ~1 week if all documents ready
